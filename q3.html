Problem: Write a program to sort a list of numbers using Bubble Sort.

arr = list(map(int, input().split()))
n = len(arr)

for i in range(n):
    for j in range(0, n - i - 1):
        if arr[j] > arr[j + 1]:
            arr[j], arr[j + 1] = arr[j + 1], arr[j]

print(arr)

Sample Input:

5 1 4 2 8

Sample Output:

[1, 2, 4, 5, 8]

Rules:

  • Use only Bubble Sort logic
  • Do not use built-in sorting functions
  • No internet or AI tools
