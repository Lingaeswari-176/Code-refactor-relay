Question 2 – Medium

Problem: Write a program to remove duplicate elements from a list without changing the order.

arr = list(map(int, input().split()))

unique = []
for i in arr:
    if i not in unique:
        unique.append(i)

print(unique)

Sample Input:

1 2 2 3 4 4 5

Sample Output:

[1, 2, 3, 4, 5]

Rules:

  • Do not use built-in functions to remove duplicates
  • Maintain the original order
  • No internet or AI tools
